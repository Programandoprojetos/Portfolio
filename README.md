# Luis Henrique | Programando Projetos

Portfólio pessoal desenvolvido para apresentar projetos autorais, habilidades em front-end, trajetória de estudos e formas de contato para sites, landing pages e interfaces.

## Acesse online

[programandoprojetos.github.io/Portfolio](https://programandoprojetos.github.io/Portfolio/)

## Projetos em destaque

- **CyberSentinel**: landing page autoral com estética tecnológica e foco visual em segurança digital.
- **FilmesFlix**: interface inspirada em streaming, criada para praticar hierarquia visual e composição moderna.
- **Portfólio Programando Projetos**: vitrine digital para apresentar trabalhos, tecnologias e evolução como desenvolvedor.
- **Studio Derenice Freitas**: sistema local para agendamentos, clientes, serviços e controle financeiro de um studio de beleza.
- **Página ChatGPT Premium**: landing page de venda com oferta, benefícios, prova social, FAQ e pedido direto pelo WhatsApp.
- **Loja Smart LV**: e-commerce com React, Node.js, autenticação, catálogo, carrinho, checkout e painel administrativo.

## O que este portfólio mostra

- Projetos reais e autorais com foco em front-end
- Seções de habilidades, experiência, serviços, estudos e contato
- Layout responsivo para desktop e mobile
- Tema claro/escuro com preferência salva no navegador
- Formulário que abre o e-mail do visitante com a mensagem pronta
- Metadados para compartilhamento em redes sociais

## Estrutura do projeto

- `index.html`: estrutura principal do site
- `style.css`: estilos, responsividade e animações
- `script.js`: interações, menu mobile, modal, formulário e efeitos
- `assets/`: favicon, imagem social e artes SVG usadas no site

## Publicação no GitHub

Este é um projeto estático. Para publicar no GitHub:

1. Crie um repositório no GitHub.
2. Envie os arquivos do projeto para a branch principal.
3. Ative o GitHub Pages ou conecte o repositório na Vercel.

## Publicação na Vercel

Na Vercel, basta importar o repositório do GitHub. Como o projeto usa HTML, CSS e JavaScript puros, não precisa de comando de build.

Configuração sugerida:

- Framework Preset: `Other`
- Build Command: deixe vazio
- Output Directory: deixe vazio
