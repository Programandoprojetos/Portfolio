# Portfólio | Programando Projetos

Portfólio pessoal desenvolvido para apresentar projetos, habilidades e formas de contato de maneira moderna, responsiva e pronta para publicação no GitHub Pages.

## Acesse online

[programandoprojetos.github.io/Portfolio](https://programandoprojetos.github.io/Portfolio/)

## O que este projeto mostra

- Projetos autorais com foco em front-end e apresentação visual
- Seções de habilidades, experiência, serviços e contato
- Layout responsivo para desktop e mobile
- Tema claro/escuro com preferência salva no navegador
- Metadados de compartilhamento para publicação

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Estrutura do projeto

- `index.html`: estrutura principal do site
- `style.css`: estilos, responsividade e animações
- `script.js`: interações, menu mobile, modal, formulário e efeitos
- `assets/`: favicon, imagem social e artes SVG usadas no site

## Publicação no GitHub Pages

Como o projeto é estático, basta enviar os arquivos para o repositório e ativar o GitHub Pages apontando para a branch principal.
